# Plataforma de Eventos e Inscripciones

API REST desarrollada con Node.js y Express para una plataforma de eventos e inscripciones.

## Temática

El proyecto representa la base inicial de una plataforma donde usuarios podrán registrarse, iniciar sesión, consultar eventos e inscribirse a actividades.

## Tecnologías

- Node.js
- Express
- dotenv
- JavaScript ES Modules
